# EDA-Project
# Exploratory Data Analysis (EDA) Project

## Project Objective
Analyze customer purchasing behavior using Python.

## Features
- Statistical Analysis
- Correlation Analysis
- Data Visualization
- Insight Generation


## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Visualizations
- Correlation Heatmap
- Distribution Plot
- Scatter Plot


## How to Run

```bash
pip install -r requirements.txt
python src/eda.py
```
